# 🎓 Student ERP Management System


## 📖 Project Overview

Student ERP Management System is a web-based application developed using **PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript**.

The system helps schools manage students, teachers, attendance, marks, assignments, fees, announcements, events, holidays, and report cards through a centralized platform.

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/sakshikadam1501-rgb/student-erp-system.git
```

### 2️⃣ Move Project

Copy the project folder to:

```text
xampp/htdocs/
```

### 3️⃣ Create Database

Create:

```text
student_erp
http://localhost:8080/phpmyadmin/index.php?route=/database/structure&db=student_erp
```

### 4️⃣ Import Database

Import:

```text
student_erp.sql
```

### 5️⃣ Configure Database

Update:

```php
config/db.php
```

with your database credentials.

### 6️⃣ Run Project

```text
http://localhost:8080/student_erp/admin/dashboard.php
```
